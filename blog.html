Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a tree-like structure representing a web page, allowing JavaScript to modify content, styles, and handle events.

How do you select an element from the DOM?

I can select DOM elements using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation lets a parent element handle events from its children, saving memory and working with new elements.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate attributes with setAttribute() and getAttribute(), and modify styles using element.style.property for dynamic CSS changes.
